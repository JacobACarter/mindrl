
.. py:function:: mindspore_rl.utils.update_config(config, env_yaml, algo_yaml)

    通过传入的yaml文件更新config。参考 `mindspore_rl/algorithm/dqn/config.py`,
    `mindspore_rl/example/env_yaml/` and `mindspore_rl/example/algo_yaml/` 获取用法。

    参数：
        - **config** (dict) - 待更新的配置。
        - **env_yaml** (str) - 环境yaml文件。
        - **algo_yaml** (str) - 算法yaml文件。
